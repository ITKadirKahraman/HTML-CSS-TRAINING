**UI Color Tokens**


***A QUICK TECHNICAL TIP:***
DonÂ´t forget the 60-30-10 rule when applying these colors:
- 60% Dominant Color (usually the background)
- 30% Secondary Color (used for balance and varietyl)
- 10% Accent Color (for Call to Actions or key details)

***GENERALLY***
```css
/* Primary */
--color-primary-1: rgb(15, 42, 68);
--color-primary-2: rgb(25, 113, 194);

/* Secondary */
--color-accent-1: rgb(255, 99, 71);
--color-accent-2: rgb(255, 165, 0);

/* Neutral */
--color-dark: rgb(33, 37, 41);
--color-light: rgb(248, 249, 250);
```
